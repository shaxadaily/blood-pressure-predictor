## Blood pressure predictor
Link to application: https://blood-pressure-predictor-ghztxydshe7qt6a6cbphis.streamlit.app

## Note: In the application you should upload the bisoprolol_final.csv file.
This file contatins the data for our machine learning model

### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
